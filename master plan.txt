# Career Assessment System V2 — Master Plan

## Vision

Transform the current aptitude test into a modern, multi-dimensional career assessment platform.

The new system should:

* feel professional
* produce more accurate recommendations
* measure multiple dimensions separately
* support adaptive AI counseling
* become the foundation of the entire platform

---

# CORE DESIGN PRINCIPLES

## 1. Separate Different Types of Assessment

The current system mixes:

* interests
* aptitude
* personality
* constraints

V2 should separate them clearly.

---

## 2. Move From Binary Choices → Scaled Responses

Current:

* forced choice
* one trait per answer

New:

* Likert scale
* weighted scoring
* multiple dimensions affected per question

Example:

"I enjoy solving complex analytical problems"

1 = Strongly Disagree
2 = Disagree
3 = Neutral
4 = Agree
5 = Strongly Agree

---

## 3. Treat Assessment Like a Real System

The goal is NOT:

* random quiz
* personality gimmick

The goal IS:

* long-term suitability analysis
* educational decision support
* realistic pathway guidance

---

# ASSESSMENT STRUCTURE

# SECTION A — INTERESTS (RIASEC)

## Goal

Measure what environments and activities the student naturally prefers.

## Dimensions

* Realistic
* Investigative
* Artistic
* Social
* Enterprising
* Conventional

## Improvements

Current:

* ~2 questions per dimension

Target:

* 6–8 questions per dimension

Total:

* ~40 RIASEC questions

## Question Style

Use:

* preferences
* enjoyment
* environment fit

Avoid:

* direct self-labeling

Good Example:
"I enjoy understanding how machines or systems work"

Bad Example:
"I am intelligent"

---

# SCORING SYSTEM V2

# CURRENT PROBLEM

Current scoring:

* single trait mapping
* equal weighting
* simplistic aggregation

---

# NEW MODEL

## Weighted Contribution

Example:

Question:
"I enjoy solving analytical problems"

Contributes:

* Investigative = +0.7
* Analytical = +0.3

---

## Normalized Scores

Convert raw scores into:

* percentages
* percentile estimates
* confidence ranges

---

## Separate Interest vs Ability

A student may:

* enjoy something
* but not currently perform well in it

The system should distinguish:

* preference
* capability
* readiness

---

# ADAPTIVE TESTING (FUTURE)

## Goal

Change question difficulty dynamically.

Example:

* strong numerical performance → harder numerical questions
* weak verbal performance → easier verbal questions

This creates:

* shorter tests
* more accurate results
* better engagement
